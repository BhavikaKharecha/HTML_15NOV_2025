Task 9: <main> Element

Question: What is the role of the <main> tag?

Answer:

  • The <main> tag is used to define the main content of a webpage.

  • Role of the <main> tag:

    • It contains the primary and most important content of the page.

    • It excludes repeated content like headers, footers, navigation, and sidebars.

    • Helps SEO and screen readers understand the main purpose of the page.

    1. <br>

Task 12: Empty Tag Example

Question: Write an example of an empty tag in HTML.

Answer:

  • Empty Tag:

    • Tags that have both an opening and a closing tag, but no content inside.

  • Example:

    • <div></div>

    • <p></p>

Task 13: Null Tag

Question: What is a null tag in HTML? Give an example.

Answer:

  • Null Tag:

    • A null tag in HTML is a tag that does not have a closing tag and cannot contain any content inside.

    • It is also called a void tag.

  • Example:

    • <br>

    • <img>

    • <hr>

Task 14: Comparison Question

Question: What is the difference between a void tag and an empty tag?

Answer:

  • Void Tag:

    • Only opening tag, no closing tag.

  • Example:

    • <br>

    • <img>

  • Empty Tag:

    • Opening + closing tag, but content is empty.

  • Example:

    • <div></div>

    • <span></span>
