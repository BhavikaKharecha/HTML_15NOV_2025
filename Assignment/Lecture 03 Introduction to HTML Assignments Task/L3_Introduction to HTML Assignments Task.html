Task 4: Match HTML version with feature Question: Match the HTML version with its feature. Ans: Match the HTML version with its featureis as follows: HTML 1.0 (1991) - The first version, designed for basic text formatting and links. HTML 2.0 (1995) - Standardized features like forms and tables. HTML 3.2 & 4.01 (1997-1999) - Introduced styles, scripting support, and multimedia. XHTML (2000) - A stricter, XML-based version of HTML. HTML5 (2014) - Modern standard supporting multimedia, semantic elements (, ), APIs, and responsive design. Task 5: Why HTML5 is important?
